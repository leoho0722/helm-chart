### Add Helm Chart Repo

```shell
helm repo add leoho0722 https://leoho0722.github.io/helm-chart
helm repo update
```

```shell
helm repo update
helm install mosquitto-test leoho0722/mosquitto-test
```

### Upgrade

```shell
helm repo update
helm upgrade mosquitto-test leoho0722/mosquitto-test
```

### Uninstall

```shell

```
